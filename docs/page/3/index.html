27 Nov, 2016 qt qml c++ programming

Today I attempted to expose a QList<int> from C++ to QML and use it as model for a repeater. The documentation here seems to indicate that a QList<int> should be treated as a JS Array in QML. This isn’t entirely true however, as I found out a QList can not be used as model, while a JS Array can. You won’t get any runtime error, the property will be read but it just won’t work.
